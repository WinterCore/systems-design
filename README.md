# Systems Design Practice

Practicing backend systems design — schema, API design, data modeling, caching, and scalability.

## Key Lessons

- Design your data model around read patterns, not write patterns.
- Precompute at write time what you'd otherwise calculate at read time.
- Start with the queries you need, then work backwards to the schema.
